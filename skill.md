## Skills are mandatory in this repo

Canonical Codex skills (standard location):

- `.codex/skills/**`

Human-readable index:

- `virric/skill.md`

Canonical agent handoff:

- `virric/AGENTS.md`

Codex skill standard:

- Each skill is a folder containing a required `SKILL.md` with YAML front matter (`name`, `description`, optional `metadata`) plus optional `scripts/`, `references/`, and `assets/`.

Reference: [Codex skills standard](https://developers.openai.com/codex/skills/).
