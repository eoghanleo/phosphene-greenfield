## D) Segmentation prioritization logic

Prioritization logic:
- Segment fit to the core loop (idle progress + sim “wow” factor)
- Market size / reachability (especially for web-first distribution)
- Monetization alignment (tolerance for fair IAP/ads, willingness to pay)

Buying-center dynamics:
- Primary buyer is the end-user player; gatekeepers include app stores / platform policies (notably for ChatGPT apps).

Deprioritized / out-of-scope segments (initially):
- Hardcore competitive/PvP-first gamers
- Players averse to insect imagery (phobia)
- Web3/NFT gaming audiences

## E) Core workflows (3–7)

Workflow list (high-level):
- W1: First session onboarding → “start colony” → first visible progress
- W2: Idle check-in loop → collect offline progress → make 1–3 impactful decisions
- W3: Challenge event → respond/mitigate → recover and learn
- W4: Meta progression → expansion/prestige → refreshed goals
- W5 (optional channel): In-chat experience → ask “why” → AI-guided adjustment (ChatGPT edition)
