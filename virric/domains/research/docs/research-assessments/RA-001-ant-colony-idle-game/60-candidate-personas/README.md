# Candidate personas (CPE)

This folder contains **Candidate Persona (CPE-####)** artifacts produced in `<research>` as **1:1 proposals** for canonical personas.

Important:
- **CPE-* are authoritative in `<research>`**.
- **PER-* are authoritative in `<product-marketing>` only**.

Create new CPE docs (preferred):

```bash
./virric/domains/research/scripts/create_candidate_persona.sh --bundle "virric/domains/research/docs/research-assessments/RA-001-ant-colony-idle-game" --name "..." --segment SEG-0001
```
